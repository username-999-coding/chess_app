# chess_app
I made my first biggest project as a new coder, which is chess.

The features:
1. Pieces moves
2. Timer
3. Resign Button
4. Move History
5. AI level / mode choice (Easy, Medium, Hard, 1v1 player)
6. Check
7. Checkmate
8. Stalemate
9. Castling (queenside and kingside)
10. En passant
11. Promotion
